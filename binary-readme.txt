【概要】

Windows 用のマンデルブロ集合描画アプリです


【機能と特徴】

* 操作レスポンス重視で作りました

* 色付けは変更できます。ランダム設定(Ctrl+R)もできます。

* マウスホイールで拡大縮小できます

* マウスドラッグでスクロールできます

* マルチスレッドで高速描画します(シングルコアでも頑張って描きます)

* 生成した画像は JPEG ファイルに保存できます

    作成した JPEG ファイルには、
    Windows エクスプローラで表示・変更可能な「コメント」プロパティに、
    画像生成時のパラメータが記録されています

    生成した画像に対する著作権は主張しません


【配布ファイルの構成】

GoldenMandel.exe - 実行ファイル

Readme.txt - このファイル

License.rtf - ライセンス概要


【インストール方法】

インストールパッケージ(.msi ファイル)を実行してください


【実行時に作成されるファイル】

配布ファイルと同一のフォルダに以下のファイルが自動的に作成されます。な
お Windows により他のディレクトリに作成される場合があります。

    GoldenMandel.ini - 環境設定ファイル

レジストリは Microsoft Windows Installer で設定される以外には使用していません


【操作方法】

* マウスホイール上回転, PageUp, マウス左ボタンダブルクリック : ズームイン(拡大)

* マウスホイール下回転, PageDown, Shift+マウス左ボタンダブルクリック, マウス右ボタンダブルクリック : ズームアウト(縮小)

    ズームは、
    マウス操作ではマウスポインタ位置を中心に、
    キーボード操作ではウィンドウ中央を中心に、
    2倍(拡大) あるいは 1/2倍(縮小) を繰り返します

* マウス左ボタンでドラッグ, スクロールバー, ←→↑↓ : スクロール(移動)

* Ctrl+R : ランダム色設定

* Ctrl+S : 名前を付けて保存


【アンインストール方法】

インストールパッケージ(.msi ファイル)で対応しています。
Windows のアプリのアンインストールを行ってください。
